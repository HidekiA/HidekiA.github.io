BrowserSwitch
[English version]
概要
メールやエクセルファイル等に記載されたURLをクリックした際に、そのURLに応じて異なるブラウザで開くためのソフトウェアです。

用途（作成理由）：
社内システムが Internet Explore (IE) にしか対応していないといった環境で、社内システム用には IE、その他のサイトはお気に入りのブラウザで開くことができます。

注意：
ブラウザ内でリンクをクリックしたときには、そのままそのブラウザで開かれます。BrowserSwitch は他のソフトウェアからブラウザを開くときにだけ使用されます。
（最近は社内ページでも Office 365 の関係で IE に完全には対応していないページがあったりするのですが、IE でそういったページへのリンクをクリックしても別のブラウザで開くことは残念ながらできません。）
ダウンロード
BrowserSwitch 2.0
インストール
  1. インストーラを実行する。
  2. Windowsの設定の「既定のアプリ」が開かれるので、「既定のブラウザー」に BrowserSwitch を選択する。
設定
実行ファイルと同じディレクトリに置かれたレシピファイル BrowserSwitchRecipe.ini に設定を記述します。
（ショートカットがスタートメニューに登録されます）
#（例）
[Rules]
intranet.co.jp=ie
www.office.com=edge
www.mozilla.org=firefox
otherwise=chrome

[Browsers]
chrome=C:\Program Files\Google\Chrome\Application\chrome.exe
firefox=C:\Program Files\Mozilla Firefox\firefox.exe
[Rules]
  • keyword=browser_name
    左辺のキーワードを含むURLを右辺のブラウザで開きます。
  • "otherwise" というキーワードは全てのURLに合致し、ローカルの html ファイルを開く場合にも使われます。
  • 大文字小文字の区別があり、空白を無視しません。
  • # で始まる行はコメントとして無視されます。
[Browsers]
  • [Rules] で指定したブラウザのパスを記述します。
  • "edge" と "ie" はブラウザ名としてあらかじめ定義されています。
アンインストール
Windows の設定の「アプリ」から BrowserSwitch を選び、アンインストールしてください。
